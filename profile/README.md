## What we build

### ATLAS — the world's first AGI experience

A continuously running, autonomous AI agent. Not a chatbot. Not a copilot. A second mind.

- **Always on.** Runs 24/7 on your machine. Wakes up with you, works while you sleep.
- **Real memory.** Remembers conversations, habits, files, people, and projects across months — not just the current session.
- **Computer use.** Sees the screen, drives a real browser, reads OCR, runs shell commands, controls the OS.
- **Builds its own tools.** When it hits something new, it writes a tool to handle it — and the next agent in the family inherits it.
- **Everywhere you are.** Telegram, WhatsApp, Discord, Slack, Matrix, email, a desktop overlay, and a native macOS app.
- **Multi-agent swarms.** Decomposes hard work across specialized sub-agents that run in parallel.
- **Deep research.** Hours-long autonomous research sessions across hundreds of sources.

## What we care about

- **AGI for everyone, not enterprises.** The default user is a single person, not a Fortune 500. Pricing, defaults, and UX follow.
- **Autonomy over assistance.** Software that takes initiative, not software that waits for instructions.
- **Memory is the moat.** A model without memory is a stranger. We build the long-term substrate that makes AI feel like *yours*.
- **Self-improvement.** Agents that learn from their own runs, write their own tools, and get better the longer they live.
- **Ship, then talk.** Working software over roadmaps and demos.
